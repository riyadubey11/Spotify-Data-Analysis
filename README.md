# Spotify Data Analysis 🎵

## Project Overview

This project analyzes Spotify song data using Python to discover patterns in music popularity, artist dominance, release trends, and song audio features.

The project applies data cleaning, exploratory data analysis (EDA), and data visualization techniques to transform raw Spotify data into meaningful insights.

## Objectives

* Analyze the most streamed songs on Spotify.
* Identify the top-performing artists.
* Study song release trends over time.
* Explore relationships between audio features.
* Understand characteristics of popular songs.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

1. Dataset Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Audio Feature Analysis
6. Insight Generation

## Visualizations Created

* Top 10 Most Streamed Songs
* Top 10 Artists by Total Streams
* Songs Released Per Year
* Correlation Heatmap
* Danceability vs Streams
* BPM Distribution
* Mood Classification

## Key Insights

* A small number of songs account for a large percentage of Spotify streams.
* A few artists dominate Spotify in terms of total streams.
* Song releases have increased in recent years.
* High-energy and highly danceable songs are common among popular tracks.
* Audio features influence listening patterns and music trends.

## Conclusion

This project demonstrates how Python can be used to clean, analyze and visualize music streaming data. The analysis provides insights into artist performance, song popularity and audio characteristics.
